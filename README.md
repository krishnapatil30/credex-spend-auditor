## 🛠️ Tech Stack & Features

### **Intelligence Engine**
- **Framework:** Next.js 15 (App Router) with Turbopack.
- **Styling:** Tailwind CSS + Shadcn/UI for a high-fidelity enterprise aesthetic.
- **Logic:** Custom audit algorithm to calculate ROI from license consolidation.

### **Data & Persistence**
- **Backend:** Supabase (PostgreSQL) for relational data storage.
- **Data Flow:** Secured via **Next.js 15 Server Actions**, reducing client-side overhead and securing API keys.
- **Schema:** UUID-based lead tracking with timestamped audit snapshots.
- **Security:** Implemented Row Level Security (RLS) to permit public inserts while protecting sensitive data.
