# AI Prompt Engineering Documentation

## Audit Summary Prompt
**Model:** `gemini-1.5-flash`
**System Instruction:**
> "You are an expert financial auditor for AI software spend at a company called Credex. Analyze the provided Team Size and Annual Savings data. Write a 100-word professional, punchy summary for a startup founder. Explain overspending via seat fragmentation and how Credex helps."

**Why this works:** By assigning a "Role" (Financial Auditor), the output becomes more authoritative. Using the "Founder" persona ensures the tone remains high-level and ROI-focused.
