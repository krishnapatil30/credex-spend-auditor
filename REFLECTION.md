# Project Reflection

## 🎯 Successes
- **Full-Stack Integration:** Successfully connected Next.js 15 Server Actions with a Supabase PostgreSQL backend.
- **AI Resilience:** Implemented a robust fallback system for the Gemini API to ensure the UI never breaks for the user.
- **Performance:** Achieved near-instant ROI visualization using CSS-based charting instead of heavy external libraries.

## 🧠 Challenges & Learning
- **Dependency Management:** Overcame a major "Workspace Root" conflict where Next.js was indexing the entire OS user directory due to a rogue lockfile.
- **Modern Next.js:** Learned to navigate the nuances of Next.js 15 and Turbopack, specifically regarding environment variable injection in production.

## 🛠️ Future Improvements
- **PDF Export:** Adding the ability for users to download their audit as a professional PDF.
- **Multi-Currency:** Support for global SaaS pricing beyond USD.
